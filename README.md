# House-Prediction-Application
 This application utilizes Flask and a Linear Regression machine learning model to predict house prices in
 Bangalore. Users input location, number of BHK, number of bathrooms, and square feet, and the model predicts
 the price with an accuracy of 80%. The application provides a user-friendly interface for users to input
 parameters and receive predicted prices, making it a valuable tool for homebuyers and real estate investors
